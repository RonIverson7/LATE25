üìö How to Use

Basic Usage: Add the base btn class plus a variant class:

  • btn btn-primary - Main action buttons
  • btn btn-museo - Museum-themed buttons
  • btn btn-ghost - Subtle action buttons

Navigation Buttons: Special classes for navigation bar:

  • nav-btn nav-btn-search - Search button
  • nav-btn nav-btn-icon - Icon buttons (notifications, etc.)
  • nav-btn nav-btn-coin - Currency/coin display
  • nav-btn nav-btn-avatar - Avatar button

More/Edit Buttons: Small buttons for options menus:

  • btn-more - Default more button (‚ãØ)
  • btn-more-dark - Dark variant
  • btn-more-ghost - Transparent variant
  • Add btn-sm or btn-lg for size variants

Sizes: Add btn-sm, btn-lg, or btn-xl

Special: Use btn-block for full width, btn-pill for rounded, btn-icon for icon-only

Dropdowns: Combine with more buttons for elegant menus:

  • dropdown - Container for dropdown
  • dropdown-menu - The menu itself
  • dropdown-item - Individual menu items
  • dropdown-separator - Visual separator
  • Add dark, sm, lg classes for variants

Modal & Request Buttons: Special buttons for admin interfaces:

  • btn-filter - Filter/tab buttons (add active for selected)
  • btn-status approved/pending/rejected/review - Status badges
  • btn-type artist-verification/content-report/feature-request - Request type badges
  • btn-modal-action approve/reject/neutral - Modal action buttons

Social Action Buttons: Elegant interaction buttons for content:

  • btn-social like/comment/share/bookmark/follow - Social interaction buttons
  • btn-social-group - Container for grouping social buttons
  • Add active for engaged state, compact/lg for sizes
  • Add icon-only for icon-only buttons
  • Use .count span for displaying interaction counts
